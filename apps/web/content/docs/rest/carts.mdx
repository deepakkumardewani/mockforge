# Carts

Access shopping cart data with items, pricing, and checkout information.

## List carts

**Endpoint:** `GET /api/carts`

**Query parameters:**

| Parameter | Type   | Example |
| --------- | ------ | ------- |
| `limit`   | number | 10      |
| `skip`    | number | 0       |
| `search`  | string | active  |
| `sort`    | string | date    |
| `order`   | string | desc    |

**Example request:**

```bash
curl "https://api.mockforge.dev/api/carts?limit=5"
```

**Example response:**

```json
{
  "data": [
    {
      "id": 1,
      "userId": 1,
      "items": [
        {
          "id": 101,
          "productId": 1,
          "quantity": 2,
          "price": 79.99
        }
      ],
      "total": 159.98,
      "discountTotal": 15.99,
      "tax": 12.8,
      "createdAt": "2024-05-12T10:30:00Z"
    }
  ],
  "pagination": {
    "limit": 5,
    "skip": 0,
    "total": 50,
    "hasMore": true
  }
}
```

## Get single cart

**Endpoint:** `GET /api/carts/:id`

**Example request:**

```bash
curl https://api.mockforge.dev/api/carts/1
```

**Example response:**

```json
{
  "id": 1,
  "userId": 1,
  "items": [
    {
      "id": 101,
      "productId": 1,
      "quantity": 2,
      "price": 79.99
    }
  ],
  "total": 159.98,
  "discountTotal": 15.99,
  "tax": 12.8,
  "createdAt": "2024-05-12T10:30:00Z"
}
```

## Search carts

**Example request:**

```bash
curl "https://api.mockforge.dev/api/carts?search=active&limit=3"
```

Searches across user-related fields.

## Create cart

**Endpoint:** `POST /api/carts`

Creates a mock cart. Does not persist.

## Update cart

**Endpoint:** `PUT /api/carts/:id`

Updates a mock cart. Does not persist.

## Delete cart

**Endpoint:** `DELETE /api/carts/:id`

Deletes a mock cart. Does not persist.
