# Quickstart

## Fetch products

```bash
curl https://api.mockforge.dev/api/products?limit=5
```

## Fetch a single user

```bash
curl https://api.mockforge.dev/api/users/1
```

## Search todos

```bash
curl "https://api.mockforge.dev/api/todos?search=grocery&limit=3"
```
