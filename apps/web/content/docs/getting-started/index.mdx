# Getting Started

No API key. No sign-up. Just send requests.

## Base URL

```
https://api.mockforge.dev
```

## Your identity

Every browser that visits MockForge gets a unique `mf_id` UUID stored in `localStorage`. This ID is sent as the `X-MF-ID` header on all requests from the schema builder UI.

For direct API access, you can pass any UUID you like:

```bash
curl -H "X-MF-ID: my-custom-id" https://api.mockforge.dev/api/products
```

## Rate limits

- **300 req/min** when `X-MF-ID` header is present
- **60 req/min** when using IP-based fallback

See [Rate Limits](/docs/rate-limits) for details.
