MockForge is a unified fake data API platform. No signup required. Just send requests.

## What's available

- **REST API** — 14 entities, full CRUD, pagination, search
- **GraphQL** — same 14 entities via a typed GraphQL schema
- **WebSockets** — live streams for stats, notifications, chat, stock ticker
- **Socket.io** — same real-time data with automatic reconnection
- **Custom Schemas** — define your own schema, get a dedicated endpoint
